# Knievel Admin UI

Operator console for the knievel ad server. Spec: `../../UI.md`.
Phase plan: `../../PHASES.md` § Phase 7.

## Quick start

```bash
pnpm install
pnpm dev          # Vite dev server on http://localhost:5173
```

The dev server expects the API at `http://localhost:8080` (knievel
default). Run `cargo run` from the repo root in another terminal.

## Per-PR gates

```bash
pnpm typecheck    # tsc -b --noEmit
pnpm lint         # eslint .
pnpm test         # vitest (run mode: `pnpm test --run`)
pnpm build        # tsc + vite build → dist/
pnpm format:check # prettier --check .
```

## What's here today

Phase 7.1 scaffold: Vite + React 18 + TypeScript, TanStack
Router (file-based) + Query, Mantine v7, ESLint/Prettier,
Vitest + Testing Library + happy-dom. One placeholder route at
`/`. No auth, no API client, no real views — those land in
later phase tasks (see `../../PHASES.md`).

## Stack reference

| Concern    | Library                                                                   |
| ---------- | ------------------------------------------------------------------------- |
| Framework  | React 18 + TypeScript                                                     |
| Build      | Vite 6                                                                    |
| Routing    | TanStack Router (file-based, plugin-generated `routeTree.gen.ts`)         |
| Data       | TanStack Query (auth + API client wired in 7.3 / 7.4)                     |
| Components | Mantine v7 (CSS modules; PostCSS preset configured)                       |
| Tests      | Vitest + Testing Library + happy-dom                                      |
| Lint       | ESLint v9 (flat config) + typescript-eslint + react-hooks + react-refresh |
| Format     | Prettier                                                                  |
| Package    | pnpm                                                                      |
